### 2. 输入框 Placeholder 提示

为 **3 个输入框** 添加了 Placeholder 提示：

#### HudTextInput 扩展
```java
// 新增字段
private String placeholder = "";

// 新增方法
public void setPlaceholder(String placeholder);
public String getPlaceholder();

// 渲染时显示 Placeholder（空白且未聚焦时）
if (display.isEmpty() && !focused && !placeholder.isEmpty()) {
    ctx.drawText(client.textRenderer, placeholder, drawX, textY, 0xFF888888, true);
}
```

#### 输入框配置
```java
// 构件名称
nameInput.setPlaceholder("输入构件名称（如：橡木门）");

// 标签
tagsInput.setPlaceholder("输入标签，用逗号分隔（如：wood, modern）");

// Socket ID
socketIdInput.setPlaceholder("输入 Socket ID（如：door_frame）");
```

**效果**：未输入内容时显示灰色提示文本，引导用户输入
