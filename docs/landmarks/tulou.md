# 土楼（福建永定）参数化地标

本文件说明 FormaCraft 当前“土楼（Tulou）”地标的可控参数、默认值与推荐提示词写法。

## 触发方式（推荐）

只要提示词包含以下任意关键词，即会优先命中**确定性模板**（而不是让大模型自由发挥）：

- `土楼` / `福建土楼` / `永定` / `tulou`

## 参数（自然语言可解析）

- **直径**：`直径为20` / `diameter 20`
  - 默认：20
  - 范围：12 ~ 80
- **层数**：`3层` / `4 floor`
  - 默认：3
  - 范围：2 ~ 6
- **门朝向**：`门朝南` / `南门` / `朝东开门` / `north/south/east/west`
  - 默认：SOUTH（南）
- **环带厚度**：`环带厚度 5` / `墙厚 4`
  - 默认：随直径自动估算（约为半径的 1/3）
  - 范围：3 ~ 8
- **内院大小**
  - **内院直径**：`内院直径 10`
  - **内院占比**：`内院占比50%` / `courtyard ratio 0.5`
  - 若同时提供：优先使用“环带厚度”，否则按内院参数推算

- **细节等级**：`更精致` / `细节更多` / `refined`
  - 默认：`aesthetic`（更观赏性）
  - 若包含“精致/细节更多/refined”等关键词：`refined`（更精致）

- **百叶窗/木窗扇（trapdoor）**：在提示词里加 `百叶窗` / `木窗扇` / `trapdoor shutter`
  - 默认：开启后在外墙窗带生成百叶窗
  - `aesthetic`：默认闭合；`refined`：默认半开（更立体）
  - 可显式指定：`百叶窗打开` / `百叶窗关闭`

## 推荐提示词示例

- `生成直径为20的福建永定土楼`
- `生成直径为24的土楼，3层，门朝南`
- `生成直径为40的福建土楼，4层，门朝东，内院占比50%`
- `生成直径为30的土楼，墙厚6，内院直径12`
- `生成直径为24的福建土楼，3层，更精致`
- `生成直径为24的福建土楼，3层，外墙窗户做木窗扇百叶（trapdoor）`

## 实现约定（面向开发者）

后端会输出：

- `footprint.shape="circle"` + `footprint.radius = 直径/2`
- `extra.landmark="tulou"`（用于稳定路由到 `TulouGenerator`）
- `extra.ringThickness / extra.doorFacing / extra.courtyardRadius / extra.courtyardRatio`（可选）
- `extra.detailLevel = aesthetic | refined`
