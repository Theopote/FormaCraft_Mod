## 明清官式院落（mingqing_courtyard）

### 1) 识别关键词（aliases）
- 明清官式院落 / 明清官式建筑群落 / 四合院 / 院落

### 2) 生成要点（v1）
- 复合体：院墙 + 门楼 + 主殿 + 左右厢房（中轴对称）
- 默认使用确定性模板以避免“语义漂移”
- 可选：院内铺路（门楼→院心→主殿，院心→厢房）

### 3) 可参数化字段（v1）
- 尺寸：`32x28`（宽×深），例如：`生成明清官式院落 40x32`
- 铺路开关：默认开启；可写 `不要路` / `不铺路` 关闭
- 路宽：`路宽 3` / `path width 3`
